Route : GET
	1. Explore -> Ambil berdasarkan new post | /posts?sort=date
	2. Search -> ambil berdasarkan name, full name | /users?username=[inp]|fullname=[inp]
	3. Post -> ambil berdasarkan following id trus ambil post yang dipost id tsb | /posts?owner=
	4. Post-Detail -> ambil berdasarkan id | /posts/[id]
	5. Saved -> ambil berdasarkan id post | /saved
	6. Commentar -> ambil berdasarkan id post | /posts/[id post]/comments
	7. Reply -> ambil berdasarkan parent(yg dikasih id comment parent)	/posts/[id post]/comments?parent=[id parent]

	POST
	1. Insert post  -> post bisa banyak img | /posts
	2. Insert comment | /comments
	3. Insert like / dislike -> brdsrkan id | /posts/[id]/likes | /posts/[id]/comments/[id post]/likes

	PATCH
	1. Update Profile | /users/[id]
	2. Update Post | /posts/[id]
	3. Update comment | /comments/[id]
	
	DELETE
	1. Del Post | /posts/[id]
	2. Del Comment | /comments/[id]
